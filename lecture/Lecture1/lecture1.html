文件命令

  • ls ：显示文件夹
  • ls -al ：显示文件夹
  • cd dir ：进入dir文件夹
  • cd ：进入home目录
  • cd - ：进入上一次进入的目录
  • pwd ：打印当前文件夹
  • mkdir dir ：创建dir文件夹
  • rm file ：删除file文件
  • rm -r dir ：递归删除文件夹
  • rm -f file ：强制删除文件
  • cp -r dir1 dir2 ：拷贝dir1到dir2

文件命令

  • cp -r file1 file2 ：拷贝file1到file2
  • mv file1 file2 :移动file1到file2
  • touch file :创建file文件
  • cat file :输出file文件
  • more file :输出file文件内容
  • head file :输出file文件的头十行
  • tail file :输出file文件的尾十行
  • ln -s file link:创建link链接文件，链接到file
