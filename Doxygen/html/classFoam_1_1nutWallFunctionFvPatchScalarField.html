Detailed Description

The class nutWallFunction is an abstract base class that hosts calculation methods and common functions for nut wall-function boundary conditions.

Usage
Example of the boundary condition specification: 
<patchName>
{
    // Top-level entries
    ...

    // Optional entries
    U             U;

    // Inherited entries
    ...
}

where the entries mean:

Property Description Type Reqd Deflt
U Name of operand velocity field word no U
