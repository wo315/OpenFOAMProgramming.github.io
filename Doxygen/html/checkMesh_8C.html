Detailed Description

Checks validity of a mesh.

Original source file checkMesh.C

Usage
checkMesh [OPTION]

Options:

  • -allGeometry
    Checks all (including non finite-volume specific) geometry
  • -allTopology
    Checks all (including non finite-volume specific) addressing
  • -meshQuality
    Checks against user defined (in system/meshQualityDict) quality settings
  • -region <name>
    Specify an alternative mesh region.
  • -allRegions
    Check all regions in regionProperties.
    Parameters
    -writeSets<surfaceFormat>
    Reconstruct all cellSets and faceSets geometry and write to postProcessing directory according to surfaceFormat (e.g. vtk or ensight). Additionally reconstructs all pointSets and writes as vtk format.
    -writeAllFields
    Writes all mesh quality measures as fields.
    -writeAllSurfaceFields
    Adds writing of surface fields when used in combination with writeAllFields.
    -writeFields'(<fieldName>)'
    Writes selected mesh quality measures as fields.
    -write-edges
    Write bad edges (possibly relevant for finite-area) in VTK format.
