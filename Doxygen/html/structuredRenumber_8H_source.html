30Description
31 Renumbering according to mesh layers.
32 depthFirst = true:
33 first column gets ids 0..nLayer-1,
34 second nLayers..2*nLayers-1 etc.
35 depthFirst = false:
36 first layer gets ids 0,1,2 etc.
