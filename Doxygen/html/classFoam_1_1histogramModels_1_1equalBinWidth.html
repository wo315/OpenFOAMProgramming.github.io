Detailed Description

Histogram model which groups data into bins of equal width.

Usage
Minimal example by using system/controlDict.functions: 
histogram1
{
    // Inherited entries
    ...

    // Mandatory entries
    nBins       <label>;

    // Optional entries
    min         <scalar>;
    max         <scalar>;
}

where the entries mean:

Property Description Type Reqd Deflt
nBins Number of histogram bins label yes -
min Minimum value of histogram data scalar no -
max Maximum value of histogram data scalar no -
Note
  • If max is not provided, histogram will use operand field's min and max as the bin extremes.
  • If max is provided without min, histogram will use 0 for the min.
