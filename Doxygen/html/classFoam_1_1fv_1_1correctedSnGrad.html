template<class Type>
class Foam::fv::correctedSnGrad< Type >

Surface gradient scheme with full explicit non-orthogonal correction.

Usage
Minimal example by using system/fvSchemes: 
snGradSchemes
{
    snGrad(<term>)       corrected;
}
