Detailed Description

A diagonal beam, which is cylinder-like, selectable as diag or diagbeam.

If the beam angle is close to zero, it will be changed to a box (PDRobstacles::cuboid)

Dictionary controls

Property Description Required Default
point The end centre-point yes
length The beam length yes
width The beam cross-dimensions yes
angle The beam angle (degrees) yes
direction The x/y/z direction yes

Example, 

diag
{
     point       (0 0 0);
     length      0.95;
     width       (0.05 0.01);
     angle       25;
     direction   x;
}
Note
Does not have porosity.
