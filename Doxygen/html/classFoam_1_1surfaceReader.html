Detailed Description

Abstract base class for surface readers with fields.

Some readers support different input options, these are typically specified as 'readOptions' in the containing dictionary.

readOptions
{
    default
    {
        verbose     false;
    }

    ensight
    {
        masterOnly  false;
    }
}
