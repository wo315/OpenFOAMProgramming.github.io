Detailed Description

template<class BasicTurbulenceModel>
class Foam::LESModels::sigma< BasicTurbulenceModel >

The sigma SGS model.

Reference: 

    Nicoud, F., Toda, H. B., Cabrit, O., Bose, S., & Lee, J. (2011).
    Using singular values to build a subgrid-scale
    model for large eddy simulations.
    Physics of fluids, 23(8), 085106.
    DOI:10.1063/1.3623274

The default model coefficients correspond to the following: 

    sigmaCoeffs
    {
        ck          0.094;
        Csigma      1.68;
    }
Note
The default value of the Csigma constant implemented was calibrated for OpenFOAM using decaying isotropic turbulence and is slightly higher than the value suggested in the reference publication.
