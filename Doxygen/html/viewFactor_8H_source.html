33Description
34 View factor radiation model. The system solved is: C q = b
35 where:
36 Cij = deltaij/Ej - (1/Ej - 1)Fij
37 q = heat flux
38 b = A eb - Ho
39 and:
40 eb = sigma*T^4
41 Ej = emissivity
42 Aij = deltaij - Fij
43 Fij = view factor matrix
